## 3. Meal Analysis
```
┌─────────────────────────────────┐
│ ← Back    Meal Analysis         │
├─────────────────────────────────┤
│                                 │
│ ┌─────┐  Arroz "A La Cubana"    
│ │🍛   │  Category: Starchy Foods 
│ │     │                         
│ └─────┘  Weight: 400g   (estimated/provided)   
│                                 
│ ┌─────────────────────────────┐ 
│ │ 🍞 CARBOHYDRATES: 70g       │ 
│ │ 💉 INSULIN: 7.0 units       │ 
│ │ ⚡ RATIONS: 7.0              │ 
│ └─────────────────────────────┘ 
│                                 │
│ Insulin Bolus Strategy:
│  (x) Standard Bolus (15m before meal)
│  ( ) Extended Bolus (____________ over __ hours)
│  ( ) Combo Bolus (__% upfront, __% over __ hours)
│                                 │
│ Choose Action:                  │
│ ┌─────────────────────────────┐ │
│ │        ACCEPT               │ │
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │       Tweak                 │ │
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │        Redo                 │ │
│ └─────────────────────────────┘ │
│  Redo Comment (mandatory)                       │
│  ┌─────────────────────────────┐ 
│  │ e.g. "Orange Juice"         │ 
│  └─────────────────────────────┘ 
│                                 │
├─────────────────────────────────┤
│  🏠   📋   💬   📊   ⚙️        │
└─────────────────────────────────┘

* **Accept:** Saves the meal analysis and selected bolus strategy to the history, then navigates to the Meal History screen (5).
* **Tweak:** Navigates to the Tweak screen (4b) to adjust the core values (weight, carbs, units) before saving.
* **Redo:** redo sends the original params, the analysis, and the redo comment. User will return to this page
```
